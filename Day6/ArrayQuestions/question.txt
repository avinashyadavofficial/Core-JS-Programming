1-5: Basic Operations
1.	Given an array const arr = [1, 2, 3, 4, 5];, return a new array where each element is squared.
2.	Check if the array ['apple', 'banana', 'cherry'] includes the word "banana".
3.	Find the index of "orange" in the array ['apple', 'banana', 'cherry', 'orange'].
4.	Create a new array that contains only even numbers from [1, 2, 3, 4, 5, 6, 7, 8].
5.	Use a built-in method to double every element in [2, 4, 6] and return the new array.
